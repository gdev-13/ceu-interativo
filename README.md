# Ceu Interativo - Hora do Dia

## 📌 Descrição

Este é um projeto web simples desenvolvido para demonstrar conceitos básicos de HTML, CSS e JavaScript. O site "Hora do Dia" exibe a hora atual e ajusta dinamicamente a imagem de fundo e a cor do plano de fundo com base no período do dia (manhã, tarde ou noite).

## 💻 Funcionalidades

- **Exibição da Hora Atual**: Mostra a hora e os minutos atuais.
- **Mudança Dinâmica de Tema**: Altera a imagem e o fundo da página dependendo da hora:
  - Manhã (6:00 - 11:59): Fundo azul claro com imagem da manhã.
  - Tarde (12:00 - 17:59): Fundo laranja/rosado com imagem da tarde.
  - Noite (18:00 - 5:59): Fundo escuro com imagem da noite.
- **Design Responsivo**: Layout centralizado e adaptável.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura da página.
- **CSS3**: Estilização e layout.
- **JavaScript**: Lógica para obter a hora atual e alterar elementos dinamicamente.

## 🚀 Como Usar

1. Clone ou baixe o repositório.
2. Abra o arquivo `index.html` em um navegador web.
3. A página carregará automaticamente e exibirá a hora atual com o tema correspondente.
